================================================================================
FLUJO DE EJECUCIÓN
================================================================================

1. Usuario ejecuta main.py
2. Se configura dependency injection
3. Se crea la ventana principal (MainWindow)
4. Usuario selecciona área (F4)
5. Usuario captura pantalla
6. Usuario extrae cédulas con OCR
7. Sistema crea sesión de procesamiento
8. Usuario inicia procesamiento
9. Sistema digita cédulas automáticamente
10. Usuario valida y presiona F2 para cada registro
11. Sistema completa sesión y genera estadísticas
